# Costco DCF Valuation Model

This project presents a full Discounted Cash Flow (DCF) valuation model built in Excel.

## Overview
The model estimates Costco’s intrinsic value by forecasting operating performance and discounting future cash flows.

## Key Features
- Cost of equity estimation using CAPM and Gordon-Shapiro
- WACC calculation based on capital structure assumptions
- Revenue growth and margin projections based on historical performance
- Free Cash Flow (FCFF) modelling
- Terminal value estimation
- Sensitivity analysis (WACC and growth rate)

## Objective
The objective is to derive a valuation range and understand the key drivers impacting the company's value.
